## 📖 How to Use

1. **Open the URL** in your browser
2. **Search** - Type keywords in the search bar
3. **Filter** - Use sidebar filters (year, source, topics)
4. **Preview** - Click any paper title to see details
5. **Download** - Click "Download" to get the PDF
6. **Export** - Select multiple papers, click "Export Selected" for BibTeX

## 🔧 Troubleshooting

**Can't access the page?**
1. Make sure FastGPT Docker container is running
2. Check that port 3000 is accessible
3. Verify the file exists: `/opt/fastgpt/4.9.8/static-assets/paper-lib/index.html`

**Papers not loading?**
- The page loads `index.jsonl` from the archive folder
- Make sure the archive directory is accessible
- Check browser console for errors (F12)

**PDFs not downloading?**
- PDFs are served from the archive folder
- Ensure the archive path is correct
- Check file permissions
