📖 How to Use

  1. Select text in any paper HTML page
  2. Copy the URL from your browser's address bar
  3. Share the link with others
  4. When they click it, the page will automatically scroll to that text and highlight it in yellow

🔧 How It Works

Modern browsers support the Text Fragments API, which allows URLs to include text directives like:

paper.html#:~:text=selected%20text

We've enhanced the pdf2htmlEX-generated HTML files with custom JavaScript that makes this work even for text on pages 2, 3, 4, etc. (which normally wouldn't work because of the custom scroll container).
